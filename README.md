# Advanced Calculator CLI App

This project is a command-line calculator built using Python.
It performs basic and advanced arithmetic operations through a menu-driven interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Power
* Square Root
* Error handling for invalid input
* Division by zero protection

## Technologies Used

* Python

## Concepts Implemented

* Functions
* Loops
* Conditionals
* Exception Handling
* User Input Handling
